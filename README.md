### Настройка сети
Для сетевого взаимодействия используется маршрутизатор Linksys WRT54GS.

Сеть: 10.0.6.0/24

Шлюз: 10.0.6.1

Адрес сервера: 10.0.6.110

Адрес камеры: 10.0.6.160

Адреса роботов: 10.0.6.101, 10.0.6.102, 10.0.6.103, 10.0.6.104, 10.0.6.105

Сетевые адреса до 10.0.6.2-10.0.6.110 отданы под статические адреса 


### Настройка роботов
1. Установка Raspberry Pi OS на CD-карты для запуска на Raspberry Pi 3.
   Установка осуществляется с помощью Raspberry Pi Imager.
2. Настройка статического IP адреса на роботе
   {Бля не помню команды}
3. Включение поддержки arduino
   {Тоже не помню, это к Артему}
5. Настройка websocket клиента на роботах
   Устанавливается пакет websocket с помощью команды:
   `apt install python3-websocket`
   В директории /home/raspberrypi{номер_робота}/Desktop помещается python скрипт connect_to_server.py для подключения к серверу по websocket и отправки команд на контроллер arduino.
   {код_клиента}
6. Подключение осуществляется запуском скрипта командой `python connect_to_server.py`. Сервер должен быть предварительно запущен.
